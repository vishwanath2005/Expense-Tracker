# Expense Tracker Application

A secure and structured **Expense & Income Management System** developed using **Spring Boot MVC**, **Thymeleaf**, and **MySQL**.  
This application enables users to record financial transactions, manage categories, and monitor their overall balance in an organized and efficient manner.

---

## Overview

The Expense Tracker application is designed to help users maintain clear visibility over their financial activities.  
It supports expense and income tracking, category-based classification, filtering, and secure authentication.

The project follows industry-standard design principles and demonstrates a clean MVC architecture.

---

## Key Features

### Authentication & Security
- User registration and login
- Session-based authentication
- Password encryption using BCrypt
- CSRF protection enabled
- Google OAuth login support (in progress)

### Expense Management
- Add, update, and delete expenses
- Categorize expenses for better organization
- Record date and time for each transaction
- User-specific expense tracking

- Add income entries

- Clean and readable expense listing interface
